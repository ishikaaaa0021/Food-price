**🍔 Food Price Analysis Dashboard**
A comprehensive interactive dashboard for analyzing food price trends, inflation rates, and category-wise price distributions using Python, Streamlit, Pandas, and Matplotlib.

**✨ Features**
Interactive Analytics Dashboard
📈 Price Trends: Visualize price movements over time with interactive line charts
📊 Statistical Analysis: Get detailed statistics including mean, median, standard deviation, and price ranges
💰 Inflation Analysis: Calculate and visualize month-over-month inflation rates for different food items
📦 Category Analysis: Compare prices across different food categories (Staple, Dairy, Protein, Produce)

**Advanced Filtering**
Filter by date range
Select specific food categories
